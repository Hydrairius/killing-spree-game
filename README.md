# Killing Spree

Competitive PvP card game — 2–4 players battle to eliminate opponents or win via Final Spree.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Features

- **Local play** with 1 human + up to 3 AI opponents
- **14 Weapons** (Melee & Ranged), **13 Utility**, **8 Special**, **8 Hidden** cards
- **Win conditions:** Eliminate all opponents OR trigger Final Spree
- **Momentum Bonus** (optional): Draw 2 + Heal 2 on eliminations
